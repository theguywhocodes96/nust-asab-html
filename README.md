## âœ¨ Features Implemented

### 1. **Basic HTML Structure**

- Proper DOCTYPE declaration
- Complete HTML5 document structure with `<html>`, `<head>`, and `<body>` tags
- Character encoding and viewport meta tags

### 2. **Page Header**

- University name as main heading (`<h1>`)
- Department name as sub-heading (`<h2>`)
- Horizontal rule separator (`<hr>`)

### 3. **Navigation Menu**

- In-page anchor links to all major sections
- Styled navigation bar with hover effects
- Quick access to: About, Programs, Research, Student Life, Admissions, and Contact

### 4. **Content Sections**

#### About ASAB

- Comprehensive description of the department
- Information about programs and focus areas

#### Image Display

- Responsive image with proper alt text
- Styled with border and auto-sizing

#### Academic Programs

- **Unordered list** for Undergraduate programs
- **Ordered list** for Postgraduate programs
- 4+ programs in each category

#### Program Details Table

- Complete table with `<thead>`, `<tbody>`, and `<tfoot>`
- Three columns: Program Name, Level, Duration
- Styled with alternating row colors

#### Research Section

- Paragraph describing research activities
- **Description List** (`<dl>`, `<dt>`, `<dd>`) for 4 research laboratories
- Detailed lab descriptions

#### Student Life

- Unordered list with 6+ activities
- Information about societies, events, workshops, and projects

#### Admissions

- Detailed admission requirements
- External link to NUST website (`target="_blank"`)
- Eligibility criteria for both undergraduate and postgraduate programs

#### Contact Information

- Table with contact details
- Email, phone, address, and office hours

### 5. **Footer**

- Project attribution note
- "Back to Top" link for easy navigation
- Copyright information

### 6. **CSS Styling**

- Professional color scheme matching NUST branding
- Responsive design with maximum width container
- Hover effects on navigation links
- Styled tables with borders and alternating colors
- Proper spacing and typography
- Visual separation between sections
